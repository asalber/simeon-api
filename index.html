SIMEON API

A Java Application Programming Interface for the Similarity Measures Ontology

View on GitHub Download .zip Download .tar.gz

SIMEON API is the official Java application programming interface for the Similarity Measures Ontology (SIMEON).

This API provides Java interfaces for comparison measures and comparison methods, in particular for similarity and dissimilarity functions.

The list of all interfaces and methods is available in the Javadoc documentation.

It also implements the main (di)similarity functions for comparing data of basic types and the different types of collections of the E-Collections ontology.
